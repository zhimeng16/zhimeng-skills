# Example: Reuse A Saved Style

用户:

```text
用赛博主神殿古典主义风格, 给我的 AI 工作流产品做一个单文件 HTML 首屏。
不要用来源品牌和原图。
```

执行:

1. 读取 `style-kits/cyber-temple-classicism/` 下全部五个文件。
2. 把用户产品名、内容、能力点映射到该风格结构。
3. 如果没有图片, 使用 CSS 占位或生成图提示词, 不热链外部资源。
4. 输出 HTML 到用户指定路径或合适项目路径。
5. 用浏览器检查桌面和移动端; 截图放 `.firecrawl/`。
6. 汇报改了什么、验证了什么、剩余风险。

验收重点:

- 第一眼是电蓝、古典神庙、铜版画、纪念碑式发布页。
- 没有第三方品牌、logo、原图、字体文件。
- 移动端不溢出。
